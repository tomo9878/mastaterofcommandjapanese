# Master of Command 日本語化Mod

Steam用ゲーム『Master of Command』を日本語でプレイするための翻訳ファイルです。  
本Modは、ゲーム内の中国語（Chinese）設定ファイルを日本語に置き換える形で動作します。

## 📥 導入方法

1. 本レポジトリから `chinese.json` をダウンロードします。
2. 以下のインストールディレクトリを開きます。
   > `C:\Program Files (x86)\Steam\steamapps\common\Master of Command\Master Of Command_Data\StreamingAssets\GameData\Languages`
3. フォルダ内にある既存の `chinese.json` を、ダウンロードしたファイルに上書き（または置き換え）してください。
   * **注意：** 念のため、元の `chinese.json` はバックアップ（コピー）しておくことを推奨します。
4. ゲームを起動し、設定（Settings/Options）から言語を **「中国語（Chinese）」** に設定してください。
5. ゲーム画面が日本語になっていれば成功です！

---

## ⚠️ 注意事項

* **非公式ファイル:** 本ファイルは非公式の翻訳です。アップデート等により動作しなくなる可能性があります。
* **言語設定:** 内部システムを書き換えるのではなく「中国語枠」を利用しているため、ゲーム内の言語選択では **「中国語」** を選ぶ必要があります。
